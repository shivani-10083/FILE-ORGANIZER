# FILE-ORGANIZER
## PROJECT DESCRIPTION
The File Organizer is a project developed to simplify the task of managing files in a computer system. In day-to-day usage, folders often become cluttered with different types of files, this project provides a solution to this problem by scanning the current working directory and sorting files into folders based on their file extensions.

The File Organizer is a project developed to simplify the task of managing files in a computer system. In day-to-day usage, folders often become overcrowded with different types of files such as documents, images, audio and videos, making it difficult to locate specific files when needed. This project provides a solution to this problem by scanning your current working directory and sorting files into appropriate folders based on their file extensions.

## OBJECTIVE
- To maintain a clean and structured directory
- To automate the process of file organization
- To reduce manual sorting of files
- To demonstrate practical use of Python file handling

## REQUIREMENTS
- Python 3.x installed  (or any other python operating platforms)
- Text editor or IDE
- Basic understanding of terminal/command prompt

## FEATURES
- Moves files into predefined folders
- Sorts files based on their extensions
- Scans the current working directory automatically
- Supports multiple file formats (text, images, audio, video)
- Easy to modify and extend

## TOOLS USED
- Spyder IDE
- Modules
- Command Line Interface

## STEPS TO INSTALL AND RUN
- Install Spyder ( or any python compiler)
- Create file name File_Organizer.py
- Write code and save it
- Execute the code

## INSTRUCTIONS FOR TESTING
- Start the Program
- Place different types of files in the project folder
- Run the Python script
- Check whether:

  Document files move to Documents folder
  
  Image files move to Pictures folder
  
  Audio files move to Music folder
  
  Video files move to Videos folder
- Verify that all files are correctly sorted

## LIMITATIONS
- Does not create folders if they are missing
- May overwrite files with the same name
- Only works for predefined file extensions
- Works only in the current directory
- No graphical user interface
